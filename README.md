# Sistema de Gestión de Cine 

## FilmaniaDB

Proyecto de **base de datos relacional** diseñado para la administración integral de un cine, permitiendo gestionar de forma eficiente películas, funciones, salas, asientos, usuarios, ventas y control de accesos mediante códigos QR.

El sistema contempla un modelo robusto que integra:

-   **Gestión de películas**: información detallada como duración, director, actores, género, clasificación, precio y tipo de película (básica, subtitulada o inclusiva).
    
-   **Cartelera y horarios**: control de funciones por fecha, sala y horario, vinculadas directamente a cada película.
    
-   **Salas y asientos**: administración de salas, capacidad, tipo y estado de los asientos para garantizar una correcta asignación.
    
-   **Venta de tickets**: emisión de tickets asociados a usuarios, asientos, horarios y sucursales, incluyendo fecha, hora y cantidad de entradas.
    
-   **Usuarios y roles**: manejo de distintos tipos de usuarios (clientes y administradores), con control de roles y permisos.
    
-   **Sucursales**: registro de múltiples sucursales con su ubicación, contacto y ventas asociadas.
    
-   **Reportes de ventas**: generación de reportes con totales por fecha para análisis administrativo.
    
-   **Códigos QR**: uso de QR para validación de tickets y control de acceso a las funciones.
    
-   **Ofertas y descuentos**: aplicación de promociones asociadas a la cartelera.
